    🗺️ 动词分类速查
    分类特征例占比
    一类动词（五段）词尾在う段，非る/非特殊書く、読む、買う、話す约70%
    二类动词（一段）词尾る，前一个假名为い段或え段食べる、見る、起きる约25%
    三类动词する（サ变）/ 来る（カ变）勉強する、来る约5%

    ⚠️ 易错：帰る(かえる)、切る(きる)、入る(はいる)、走る(はしる) 虽然以"る"结尾但属于一类动词

    第一章 活用（词类变形）

    第一节 一类形容词活用

    📋 一类形容词敬体
    形式规则例（楽しい）特殊（いい）
    非过去肯定〜いです楽しいですいいです
    非过去否定〜くないです楽しくないですよくないです
    过去肯定〜かったです楽しかったですよかったです
    过去否定〜くなかったです楽しくなかったですよくなかったです
    📋 一类形容词简体
    形式规则例（楽しい）特殊（いい）
    非过去肯定〜い楽しいいい
    非过去否定〜くない楽しくないよくない
    过去肯定〜かった楽しかったよかった
    过去否定〜くなかった楽しくなかったよくなかった
    📋 一类形容词其他变形
    形式规则例（楽しい）特殊（いい）
    て形〜くて楽しくてよくて
    假定形〜ければ楽しければよければ
    修饰名词〜い + N楽しい一日 / いい天気—
    修饰动词〜く + V楽しく遊ぶ / よく考える—
    名词化〜さ楽しさよさ

    第二节 二类形容词活用

    📋 二类形容词敬体
    形式规则例（静か）特殊（きれい）
    非过去肯定〜です静かですきれいです
    非过去否定〜ではありません静かではありませんきれいではありません
    过去肯定〜でした静かでしたきれいでした
    过去否定〜ではありませんでした静かではありませんでしたきれいではありませんでした
    📋 二类形容词简体
    形式规则例（静か）特殊（きれい）
    非过去肯定〜だ静かだきれいだ
    非过去否定〜ではない静かではないきれいではない
    过去肯定〜だった静かだったきれいだった
    过去否定〜ではなかった静かではなかったきれいではなかった
    📋 二类形容词其他变形
    形式规则例（静か）例（きれい）
    て形〜で静かできれいで
    假定形〜なら(ば)静かなら(ば)きれいなら(ば)
    修饰名词〜な + N静かなところきれいなところ
    修饰动词〜に + V静かに読むきれいに掃除する
    名词化〜さ静かさきれいさ

    ※ 注意：单词中最后一个假名是"い"的常见二类形容词包含：きれい、嫌い、幸い

    第三节 名词的使用

    📋 名词敬体/简体活用 — 名词本身无活用，但在句型中使用方式类似二类形容词
    形式规则例（学生）
    敬体非过去肯定〜です学生です
    敬体非过去否定〜ではありません学生ではありません
    敬体过去肯定〜でした学生でした
    敬体过去否定〜ではありませんでした学生ではありませんでした
    简体非过去肯定〜だ学生だ
    简体非过去否定〜ではない学生ではない
    简体过去肯定〜だった学生だった
    简体过去否定〜ではなかった学生ではなかった
    て形〜で学生で
    假定形〜なら(ば) / であれば学生なら(ば) / 学生であれば

    第四节 动词活用

    ⏰ 动词时态一览
    形式规则例（書く）
    简体非过去肯定Vる書く
    简体非过去否定Vない書かない
    简体过去肯定Vた書いた
    简体过去否定Vなかった書かなかった
    敬体非过去肯定Vます書きます
    敬体非过去否定Vません書きません
    敬体过去肯定Vました書きました
    敬体过去否定Vませんでした書きませんでした
    修饰名词V辞書形/た形 + N書く時、書いた時、書かない時
    ✏️ ます形（动词连用形）
    词类规则原形 → ます形特殊
    一类动词（五段）う段 → い段 + ます書く → 書きます—
    二类动词（一段）る + ます食べる → 食べます—
    三类动词する → します勉強する → 勉強します—
    三类动词来る → 来(き)ます来る → 来ます—
    🔗 て形
    词类规则例特殊
    一类 う・つ・る＋って買う→買って / 待つ→待って / ある→あって行く→行って
    一类 む・ぶ・ぬ＋んで読む→読んで / 遊ぶ→遊んで / 死ぬ→死んで—
    一类 く＋いて書く→書いて—
    一类 ぐ＋いで泳ぐ→泳いで—
    一类 す＋して話す→話して—
    二类る → て食べる→食べて—
    三类 する→ して勉強する→勉強して—
    三类 来る→ 来(き)て来る→来て—

    ※ 补充：一类形容词て形 = い→くて（楽しい→楽しくて）；二类形容词て形 = 词干+で（静か→静かで）；名词て形 = N+で（学生→学生で）

    ⏪ た形
    词类规则例
    一类 う・つ・る＋った買う→買った

    第二章 助词（助詞）

    に — 用法总结

    ① 动作进行的时间点（带数字的时间点 / 特殊时间点如"夏休み""誕生日"）

    毎日6時に起きる。（每天6点起床。）

    午後3時に出かけた。（下午3点出了门。）

    ② 提示对象（动作的对象/存在的地点）

    先生は学生に日本語を教える。（老师教学生日语。）

    公園に木がある。（公园里有树。）

    ③ 提示频率 — 句型：N(期间)にN(次数)回

    一か月に二回映画を見る。（一个月看两次电影。）

    ④ 提示目的 — 句型：Vます形/N + に行く/来る/帰る

    神戸へ日本料理を食べに行く。（去神户吃日本料理。）

    ⑤ 被动句中提示施动者

    私は先生に褒められた。（我被老师表扬了。）

    ⑥ 表示目的/用途/评价 — 接续：Vる + のに / N + に

    このはさみは花を切るのに使う。（这把剪刀用来剪花。）

    ⑦ 提示目的地/终点（常与行く/来る/入る等连用）

    ⑧ 提示动作着落点（常与座る/乗る/登る/着く等连用）

    ここに住所を書いてください。（请在这里写下住址。）

    ⑨ 提示搭配内容 — 句型：N1にN2

    ⑩ 提示原因（多用于心理、情感动词：苦しむ/困る/疲れる/驚く等）

    人間関係に疲れる。（疲于人际交往。）

    ⑪ 提示能力主体 — 句型：Nには可能动词

    私には理解できない。（我无法理解。）

    で — 用法总结

    ① 表示手段/方式/工具/材料

    日本語で手紙を書く。（用日语写信。）

    この家は木でできた。（这个房子由木头建成。）

    ② 提示动作发生场所

    駅で新聞を買う。（在车站买报纸。）

    ※ 句型"N场所で〜がある/起こる"表示在某地发生某事

    ③ 表示并列（中顿） — 句型：N/A2 + で

    奈良は静かで、きれいな町だ。（奈良是安静又漂亮的城市。）

    ④ 提示数量/范围 — 句型"N数量词+で"

    二人で行ったほうが安心だ。（两个人去比较安心。）

    今日で20歳になった。（今天20岁了。）

    ⑤ 表限定（时间/金钱/数量）

    駅まで30分で行けるか。（30分钟能到车站吗？）

    を — 用法总结

    ① 提示他动词宾语 — 句型：NをV他

    ご飯を食べる。（吃饭。）

    ② 提示离开/出发的场所

    7時に家を出る。（7点出家门。）

    梅田で電車を降りる。（在梅田下车。）

    ③ 提示动作经过的场所 — 句型：N地点をV移动

    公園を散歩する。（在公园散步。）

    鳥は空を飛ぶ。（鸟儿飞过天空。）

    常用移动动词：歩く/走る/泳ぐ/渡る/散歩する/飛ぶ/登る/通る

    ④ 提示使役对象的自动词句

    母は子供を外で遊ばせる。（妈妈让孩子到外面玩。）

    子供は母を心配させる。（孩子让妈妈担心。）

    は — 用法总结

    ① 提示主题（提示主语/宾语作主题）

    ② 表示对比

    昨日は山が見えたが、今日は見えない。（昨天能看到山，今天看不到。）

    李さんは学生だが、王さんは会社員だ。（小李是学生，小王是公司职员。）

    ③ 表示最小数量（多译为"至少"） — 句型：N数量词+は

    この車を買うのに300万円は要る。（买这辆车至少要花300万日元。）

    ④ 宾语主题化（把"を"变成"は"提到句首）

    荷物はここに置かないでください。（行李请不要放在这里。）

    ⑤ 强调否定（加在其他助词后面表强调，但不能加在が/を/に/へ/と后）

    が — 用法总结

    ① 描述客观现象（自然现象、视觉/听觉发现）

    雨が降ります。（下雨。）

    ② 提示疑问词主语（が问が答）

    誰がいますか。→ 李さんがいます。（"谁在？""小李在。"）

    ③ 提示定语从句中的小主语（可用"の"替换）

    これは田中さんが作ったケーキです。（这是田中做的蛋糕。）

    ④ 提示喜好/能力/知觉/欲求/所有

    イタリア料理が好きだ。（喜欢意大利料理。）

    日本語ができる。（会日语。）

    私は犬が怖い。（我怕狗。）

    搭配词汇：わかる/できる/上手/下手/好き/嫌い/ほしい/いる/ある/怖い/心配 等

    ⑤ 表听觉/嗅觉/肌体感觉 — 句型：N + がする

    この料理はいい匂いがする。（这料理有股香味。）

    ⑥ 描述整体属性 — 句型：N1はN2が〜

    大阪は食べ物がおいしいです。（大阪的食物很好吃。）

    ⑦ 句末委婉语气（无需翻译）

    ⑧ 句末表转折（"虽然…但是…"）

    から — 用法总结

    ① 表示时间/空间的起点

    9時から5時まで勉強します。（从9点学习到5点。）

    ② 表示对象（多指动作发出者/来源）

    私は山田さんから日本語を習います。（我跟山田学日语。）

    ③ 表示原因（客观原因）

    時間がありませんから、新聞を読みません。（因为没时间，不看报纸。）

    ④ 表示原料

    ビールは麦から造られます。（啤酒由小麦酿造。）

    ⑤ 表示思考/判断的依据

    試験結果から判断する。（从考试结果来判断。）

    常见句型：から見ると/から見れば/から見て（从…来看）、から言うと/から言えば/から言って（从…来说）、からすると/からすれば/からして（从…来看）

    まで — 用法总结

    ① 表示时间/空间的终点

    12時まで勉強する。（学习到12点。）

    友達を駅まで送る。（把朋友送到车站。）

    ※ 注意："10時までです"和"10時に終わります"正确，但"10時まで終わります"是错的

    ② 表示某个时间段内持续的动作/状态

    去年までこの会社で働いていた。（直到去年都在这个公司工作。）

    ③ 表极端举例（"甚至…，连…也"）

    親友まで私のことを信じてくれなかった。（连好友都不相信我。）

    常见句型：N1はもちろん、N2も/まで/さえ…（N1就不用说了，连N2都…）

    ④ Vるまでもない（没必要/用不着）

    も — 用法总结

    ① 表示"也"

    李さんは学生だ。王さんも学生です。（小李是学生。小王也是。）

    常见句型：〜も〜も（并列）、〜も〜ば〜も（并列）

    ② 表示完全肯定/完全否定 — 句型：N疑问词+も

    どちらも好きです。（两个都喜欢。）

    何も食べません。（什么都不吃。）

    ③ 主观认为数量多（"竟然…"） — 句型：N数量词+も+肯定

    この車を買うのに400万円も要りますか。（买这辆车竟然要花400万日元吗。）

    ④ 主观认为数量少 — 句型：N最小数量词+も+否定

    教室には一人もいません。（教室里一个人都没有。）

    の — 用法总结

    ① 提示所属/性质/方位等关系

    ② 代替名词/省略后面的名词（只能省略物品，不能省略人）

    ③ 名词化（Vる + の）

    テニスをするのが好きです。（喜欢打网球。）

    ④ 提示定语从句中的主语（替换が）

    ⑤ 句末解释说明/疑问（〜のです / 〜んです）

    と — 用法总结

    ① 提示变化的结果

    日本語教師となりました。（成为了日语教师。）

    ② 提示比较的对象

    ③ 表引用内容（常接：信じる/考える/思う/言う 等）

    明日雨が降ると思います。（我认为明天要下雨。）

    ④ 并列（AとB）

    銀行の休みは土曜日と日曜日です。（银行休息日是周六和周日。）

    へ — 用法总结

    ① 提示移动方向

    東京へ行く。（去东京。）

    ② 提示传递的对象

    父への手紙（给父亲的信）

    より — 用法总结

    ① 提示比较基准 — 句型：AはBより（A比B更…）

    私より李さんは背が高い。（比起我，小李身高更高。）

    この車はその車よりずっと速いです。（这辆车比那辆车快得多。）

    ② 表示"更加"（副词用法）

    より深く理解する。（更深的理解。）

    か — 用法总结

    ① 表示选择（"或者"）

    私か妹が行きましょう。（我或者我妹妹去吧。）

    ② 表示不确定 — 句型：N疑问词+か

    誰か教室にいますか。（有人在教室吗？）

    ③ 句末疑问

    ④ 接续简体句成为主句一部分（帯疑问词简体句+か / 无疑问词简体句+かどうか）

    でも — 用法总结

    ① 表示"无论…" — N疑问词+でも

    何でも / 誰でも / いつでも / どこでも（无论什么/谁/何时/何地）

    ② 表示转折（"即使…"）

    雨でも出かける。（即使下雨也要出门。）

    常用搭配：たとえ〜でも/ても（即使…也…）

    ③ 表示举例（"…之类的"）

    お茶でも飲みながら…（一起边喝点茶什么的边…）

    ④ 极举例（"连…都"）

    子供でも分かります。（连孩子都懂。）

    ぐらい / くらい — 用法总结

    ① 表示大概数量 — N数量词+ぐらい/くらい

    会社まで一時間ぐらいかかります。（去公司大概花一个小时左右。）

    ② 表极端程度（最低/最高程度均可）

    部屋ぐらい自分で片付けなさい。（至少自己收拾一下房间。）

    涙が出るくらい嬉しい。（高兴得流泪。）

    だけ — 用法总结

    ① 表示限定（"只"）

    日本語の本は一冊だけ残っています。（日语书只剩一本。）

    常见句型：だけではなく（不仅是…）

    ② 表示最大限度/范围（"只要是…""尽可能地"）

    好きなだけお取りください。（您喜欢的请随意取用。）

    しか — 用法总结

    ① 表示限定（"只"）— 必须接否定表达

    1時間しか待てません。（只能等一个小时。）

    ② 别无他法 — Vる+しかない

    高すぎて買えないから、借りるしかない。（太贵买不起，只能借。）

    ばかり — 用法总结

    ① 表示"光是/尽是/只是"

    肉ばかり食べてはだめよ。（光吃肉是不行的哦。）

    ② 总在做某事（有责备语气）— Vてばかりいる

    弟はゲームをしてばかりいる。（弟弟总在玩游戏。）

    ③ 刚做完某事（主观认为时间短）— Vたばかりだ

    私は先月卒業したばかりです。（我上个月刚毕业。）

    "Vたばかりだ"强调主观认为时间短；"Vたところだ"强调客观时间短

    ④ 表示不好的趋势（"越来越…"）— Vるばかりだ

    英語の成績は下がるばかりだ。（英语成绩不断下降。）

    さえ — 用法总结

    ① 极举例（"连…都"）

    この本は子供でさえ読める。（这本书连孩子都能读。）

    ② 只要…就 — Nさえ〜ば

    君さえいれば十分だ。（只要你在就足够了。）

    暇さえあれば、本を読んでいる。（只要有空就在看书。）

    第三章 句型（文型）

    第一节 动词变形常考句型

    Vることができる / V可能形 — "能…"

    お酒が飲めません / お酒を飲むことができません。（喝不了酒。）

    V可能形 = 一类う段→え段+る；二类る+られる；三类する→できる；来る→来られる

    被动句型总结

    A. 直接被动 — N承受者はN施动者にV(ら)れる

    私は母に買い物を頼まれました。（我被妈妈委托去购物。）

    B. 无施动者被动 — 主语是无生命物体，不在乎谁做的

    この建物は1999年に建てられました。（此建筑建于1999年。）

    C. 创造发明类被动 — 施动者用"によって"提示

    この空港は有名な建築家によって設計されました。（这个飞机场由名家设计。）

    D. 间接被动（受害被动）— 动作不直接针对主语却连累主语

    私は隣の人にたばこを吸われました。（隔壁人抽烟影响到了我。）

    E. 所有物被动 — 主语的所有物被损害

    私は母に日記を見られました。（我的日记被妈妈看了。）

    F. 尊敬被动 — 不能用于自己/己方

    先生は毎日新聞を読まれています。（老师每天都看报纸。）

    使役句型总结

    A. 自动词使役 — N主语はN被使役者をV自(さ)せる

    私は子供を外で遊ばせます。（我让孩子到外面玩。）

    B. 他动词使役 — N主语はN被使役者にN宾语をV他(さ)せる

    私は子供に野菜を食べさせます。（我让孩子吃蔬菜。）

    ※ 他动词使役句中无论宾语是否出现，被使役者都用"に"提示

    使役 + 授受 组合句型
    意思句型例
    允许别人做V(さ)せてあげる/やる日本語の本を読ませてあげる（允许小李看日语书）
    请求允许自己做V(さ)せてもらえませんか日本語の本を読ませてもらえませんか（可以让我看日语书吗？）
    请求对方允许自己做V(さ)せてください/くれませんか日本語の本を読ませてくださいませんか（可以让我看日语书吗？）

    第二节 自他动词 / 意志无意志 / 瞬间持续动词

    自他动词常考句型

    核心区别：

    自动词他动词
    主语标记Nが + V自Nは + Nを + V他
    例句ドアが開く（门开了）ドアを開ける（开门）

    常见自他动词对：

    自动词他动词规律
    開く(あく)開ける(あける)自=あ段+る VS 他=え段+る
    閉まる(しまる)閉める(しめる)自=あ段+る VS 他=え段+る
    止まる(とまる)止める(とめる)自=あ段+る VS 他=え段+る
    始まる(はじまる)始める(はじめる)自=あ段+る VS 他=え段+る
    変わる(かわる)変える(かえる)自=あ段+る VS 他=え段+る
    消える(きえる)消す(けす)自=え段+る VS 他=す
    出る(でる)出す(だす)自=え段+る VS 他=す
    入る(はいる)入れる(いれる)特殊
    出る(でる)出す(だす)自=え段+る VS 他=す
    切れる(きれる)切る(きる)自=え段+る VS 他=う段
    届く(とどく)届ける(とどける)自=く VS 他=ける
    つくつける自=く VS 他=ける
    かかるかける特殊

    规律总结：
    规律一：自动词（あ段+る结尾）VS 他动词（え段+る结尾）
    规律二：自动词（く结尾）VS 他动词（す结尾）
    规律三：自动词（う段结尾）VS 他动词（え段+る结尾）

    第三节 ～する・なる 相关句型

    〜にする / 〜になる

    〜にする = 主观选择/决定（人为使其变成）

    コーヒーにします。（我要咖啡。/ 我决定要咖啡。）

    〜になる = 客观变化（自然而然变成）

    もう春になりました。（已经是春天了。）

    日本語教師になりました。（成为了日语教师。）

    A2/N + にする（使…变得…）

    部屋をきれいにしました。（把房间打扫干净了。）

    A2/N + になる（变得…）

    部屋がきれいになりました。（房间变干净了。）

    A1くする / A1くなる

    音を大きくする。（把声音调大。）→ 人为

    音が大きくなる。（声音变大了。）→ 自然

    第四节 否定表达常考句型

    否定表达系列

    Vなくてはいけない / Vなければいけない / Vないといけない — "必须…"

    野菜を食べなければいけない。（必须吃蔬菜。）

    Vなくてもいい — "不…也行"

    明日来なくてもいいです。（明天不来也行。）

    Vてはいけない — "不可以…"

    ここでタバコを吸ってはいけない。（不可以在这里抽烟。）

    Vなくちゃ（口语）= Vなければ

    第五节 助动词常用句型① "想"系列

    希望・愿望表达

    Vます形 + たい — "想做…"（第一人称）

    日本へ行きたいです。（想去日本。）

    Vます形 + たがる — "想做…"（第三人称）

    弟は日本へ行きたがっている。（弟弟想去日本。）

    N + がほしい — "想要…"

    新しいパソコンがほしい。（想要新电脑。）

    Vてほしい — "希望别人做…"

    早く来てほしい。（希望你早点来。）

    第六节 助动词常用句型② "好像"系列

    推量・样态表达

    Vます形 + そうだ — "看起来要…"（样态）

    雨が降りそうです。（看起来要下雨。）

    A1い → 〜そうだ（去い）

    このケーキはおいしそうです。（这个蛋糕看起来很好吃。）

    简体句 + ようだ — "好像…"（主观推断/比喻）

    简体句 + みたいだ — "好像…"（口语，= ようだ）

    简体句 + らしい — "据说…"/"好像…"（客观传闻/典型特征）

    简体句 + そうだ — "听说…"（传闻）

    第七节 常用接续句型

    常用接续句型一览

    Vて / A1くて / A2で / Nで — 并列/顺序/原因

    朝起きて、顔を洗って、ご飯を食べる。（早上起床、洗脸、吃饭。）

    Vたり Vたりする — "又…又…"（举例）

    日曜日は本を読んだり、映画を見たりする。（周日读读书，看看电影。）

    Vながら — "一边…一边…"

    音楽を聞きながら歩く。（边听音乐边走。）

    V前に / Vた後で — "在…之前/之后"

    Vと — "一…就…"（必然结果）

    Vたら — "如果…就…"（假定/发现）

    Vば — "如果…"（假设/条件）

    Vても — "即使…也…"

    Vし — 列举理由（多个原因之一）

    Vので — "因为…所以…"（客观因果）

    Vから — "因为…"（主观因果）

    Vために — "为了…"（目的）/ "因为…"（原因）

    Vことにする / Vことになる — "决定做…"（主观/客观）

    第八节 补助动词常考句型

    补助动词用法

    Vている — 正在进行/状态持续/反复习惯

    今、ご飯を食べている。（现在正在吃饭。）

    結婚している。（已婚。）

    Vてある — 动作结果的状态（人为准备）

    窓が開けてある。（窗户是开着的。— 有人打开的）

    对比：窓が開いている（窗户开着。— 不强调人为）

    Vておく — 事先做好

    Vてしまう — 完了/遗憾（口语：ちゃう/じゃう）

    Vてみる — 尝试做…

    Vていく / Vてくる — 时空趋势（去/来）

    第九节 授受表达（授受表現）

    あげる / くれる / もらう
    方向动词句型含义
    我→别人あげる/やるVてあげる我为别人做…
    别人→我くれるVてくれる别人为我做…
    我←别人もらうVてもらう我请别人做…/从别人那里得到…

    敬语版：あげる→差し上げる / くれる→くださる / もらう→いただく

    第十节 敬谦表达

    敬语三种类型

    尊敬语（抬高对方）：お/ご + Vます + になる / V(ら)れる / 特殊尊敬动词

    谦让语（贬低自己）：お/ご + Vます + する / 特殊谦让动词

    郑重语（礼貌）：です/ます / ございます

    常见特殊动词：言う→おっしゃる(尊)/申す(谦)、行く/来る→いらっしゃる(尊)/参る(谦)、食べる/飲む→召し上がる(尊)/いただく(谦) 等

    こと / もの / の / ところ / はず / わけ

    こと — 将动词名词化（抽象/一般性）

    もの — 将动词名词化（具体/本质）

    の — 将动词名词化（具体/个别）

    ところ — 时间场合

    はず — "应该…"（客观推断）

    わけ — "当然…"/"并不是…"（わけではない）

    つもり — "打算…"

    まま — "保持某种状态…"
